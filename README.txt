# DockerExam
How to work on this WebApp:

First of all, clone the  repository:
	git clone https://github.com/gurnav/DockerExam.git

And go inside it:
	cd DockerExam/

Then install the environment:
	docker-compose up -d

Now the webserver is running see the home page at:
	YourIPAddress:8080
	For MAC the IPAddress is localhost.

The phpmyadmin should be at:
	YourIPAddress:8081
	For MAC the IPAddress is localhost.

The file directory of the server is:
	DockerExam/src/
